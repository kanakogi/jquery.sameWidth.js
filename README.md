# jquery.sameWidth.js

指定したHTML要素の幅を同じにします。

## Usage


### All elements width same.

#### HTML


	<div class="same">hoge</div>
	<div class="same">hoge</div>
	<div class="same">hoge</div>
	<div class="same">hoge</div>			



#### Script
		
	<script>
	  jQuery(function($){
	    $('.same').sameWidth();
	  });
	</script>
